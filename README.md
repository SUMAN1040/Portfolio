# 🚀 Advanced 3D Interactive Portfolio

A premium, high-performance portfolio website built with modern web technologies, featuring immersive 3D interactions, glassmorphism aesthetics, and a developer-centric user experience.

![Portfolio Preview](https://via.placeholder.com/1280x720.png?text=Premium+3D+Portfolio+Mockup) 

## ✨ Key Features

- **🛡️ Engineer-Style Splash Screen**: A high-tech "Welcome" entry with terminal prompt, typewriter effect, and monospaced typography.
- **✨ Interactive 3D Cards**: Advanced tilt effects on project and skill cards that respond to mouse movements.
- **🌌 Particle Background**: Dynamic, high-performance canvas-based particle system for an immersive atmosphere.
- **💎 Glassmorphism Design**: Sleek, modern UI utilizing translucency, blurs, and vibrant gradients.
- **📱 Fully Responsive**: Optimized for seamless viewing on Desktop, Tablet, and Mobile devices.
- **📖 3D Experience Book**: Unique interactive book component for showcasing career milestones.
- **✉️ Functional Contact Form**: Integrated with Formspree for AJAX-based email delivery with real-time validation.

## 🛠️ Tech Stack

- **Core**: HTML5, Vanilla CSS3, Modern JavaScript (ES6+)
- **Icons & Fonts**: [Font Awesome](https://fontawesome.com/), [Google Fonts](https://fonts.google.com/) (Outfit & Fira Code)
- **Logos**: [DevIcon](https://devicon.dev/)
- **Backend Service**: [Formspree](https://formspree.io/) (Email Handling)

## 🚀 Getting Started

To run this project locally:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/SUMAN1040/Portfolio.git
   ```

2. **Navigate to the directory**:
   ```bash
   cd Portfolio
   ```

3. **Open in Browser**:
   Simply open `index.html` in your favorite browser or use a Live Server extension in VS Code.
